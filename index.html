Minggu 1

Minggu 1 membahas mengenai pengenalan konsep SIG berbasis Web yang memuat mengenai bagaimana internet bekerja serta instalasi Ubuntu dan WSL

Minggu 2

Minggu 2 mengenai pembuatan laman dasar menggunakan HTML dan CSS. Minggu ini mahasiswa mempelajari mengenai pembuatan web dasar dengan syntax-syntax sederhana.

Minggu 3

Minggu 3 mempelajari tentang client based scripting termasuk didalamnya adalah pembuatan fungsi-fungsi untuk perhitungan sederhana dengan script JavaScript yang tertanam di HTML

Minggu 4

Minggu 4 membahas mengenai framework bootstrap yang memiliki basis CSS dan dapat digunakan untuk mempermudah workflow pembuatan website yang rapi dan cantik.

Minggu 5

Minggu 5 mempelajari mengenai dasar Leaflet sebagai salah satu framework yang dapat digunakan untuk menyajikan informasi spasial pada website yang dibuat.

Minggu 6

Minggu 6 mempelajari Leaflet lebih lanjut dimana mahasiswa dapat menginput data spasial berupa batas wilayah dalam format GeoJSON untuk ditampilkan di website.
